# VFS Explorer

Virtual File System Explorer developed in C++ using advanced Data Structures and Algorithms.

## Features

- Virtual File System
- N-ary Tree Directory Structure
- DFS Search
- BFS Search
- Undo / Redo
- Trie Based Auto Completion
- Expression Trees
- Workspace Persistence
- Command History
- Bookmarks

## Data Structures

- N-ary Trees
- Binary Trees
- Tries
- Stacks
- Queues

## Technologies

- C++
- STL
- Windows Console API
